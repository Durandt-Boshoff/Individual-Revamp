# Individual-Revamp

What I learned:

This project helped me develop a much better understanding of Bootstrap and how it can be combined with custom CSS to create a modern, responsive website. Throughout the redesign of the NetCafé website, I learned how to structure multiple pages.

During the project I learned how Bootstrap's responsive grid automatically adjusts layouts for different screen sizes. Instead of positioning elements manually, I used rows and columns to organise text, images and forms in a clean and responsive way. I also learned how utility classes such as spacing, alignment, typography and shadows can quickly improve the appearance of a webpage while reducing the amount of custom CSS that needs to be written.

Another important skill I developed was combining Bootstrap with my own CSS. Rather than accepting Bootstrap's default colours and styling, I customised buttons, navigation links, cards, forms, backgrounds and the footer to match the warm colour palette of the NetCafé brand. I also learned how CSS specificity works when overriding Bootstrap styles and how to apply background images, gradients, hover effects, transitions and custom colour schemes to create a more professional design.

I also gained experience working with different types of website content. I embedded a PDF menu using an iframe, added downloadable files, created contact forms, used Font Awesome icons to improve visual communication and organised content into sections that were easy to read. Throughout the project I frequently tested my pages, fixed broken links, corrected file paths, resolved styling conflicts and debugged issues when content did not display correctly. This improved my troubleshooting skills and showed me the importance of testing websites during development.

Overall, this project strengthened both my HTML and CSS knowledge while giving me valuable experience using Bootstrap to create responsive, user-friendly websites. I learned that good web design is not only about making a website attractive but also about creating clear navigation, consistent layouts and an enjoyable user experience. By the end of the project, I felt much more confident using Bootstrap components together with custom CSS to redesign an existing website into something more modern and visually appealing.
